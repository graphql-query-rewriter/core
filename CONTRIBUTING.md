## Instructions

These steps will guide you through contributing to this project:

- Fork the repo
- Clone it and install dependencies

Make and commit your changes. Make sure the commands yarn build and yarn test are working.

Finally send a [GitHub Pull Request](https://github.com/chanind/graphql-query-rewriter/compare?expand=1) with a clear list of what you've done (read more [about pull requests](https://help.github.com/articles/about-pull-requests/)). Make sure all of your commits are atomic (one feature per commit).
